# Code of Conduct

This project follows the [Contributor Covenant v2.1][cov].

In short: treat other contributors with respect. Disagreement is fine —
disrespect is not. The full text of the Covenant defines expected
behavior, unacceptable behavior, scope, and enforcement.

## Reporting

Report Code of Conduct concerns to the maintainer via the email listed
on the [GitHub profile](https://github.com/Arsenalist). Reports are
handled privately.

Maintainers will review and respond. Consequences range from a private
warning to a permanent ban from project spaces, depending on severity
and pattern.

[cov]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
